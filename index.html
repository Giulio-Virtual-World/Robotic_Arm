Braccio Robotico Stampato in 3D e Controllato Multi-Piattaforma

Un sistema che combina modellazione e stampa 3D, Arduino, Node.js e interfacce web e Discord per il controllo remoto di un braccio robotico completamente realizzato con la stampa 3D.

Sistemi di Controllo Integrati

Il braccio robotico pu√≤ essere controllato attraverso tre metodi diversi, che operano in modo indipendente o combinato:

Controllo Joystick

Controllo diretto tramite joystick analogico collegato ad Arduino. Il joystick gestisce:

  • Asse X: Movimento simultaneo di spalla e gomito
  • Asse Y: Rotazione della base
  • Pulsante: Apertura/chiusura pinza

Controllo Discord

Controllo remoto tramite bot Discord che interpreta comandi testuali:

  • Formato messaggi:
    [motore][gradi]
    1360 ‚ûî Ruota la base di 360¬∞
    3-45 ‚ûî Muovi il gomito di -45¬∞
  • Bot (Node.js): Riceve comandi dal canale Discord e li invia ad Arduino via seriale
  • Feedback: Risposte in tempo reale sul canale

Controllo Web

Interfaccia grafica web che invia comandi tramite webhook Discord:

  • Interfaccia: Controlli visivi per ogni motore
  • Webhook: Invia messaggi al canale Discord
  • Sequenze: Programmazione di movimenti complessi
  • Salvataggio: Esportazione/importazione sequenze

Flusso dei Dati

Il sistema opera su due canali principali:

  1. Canale Web/Discord (Controllo Remoto)

    Interfaccia Web ‚Üí Webhook Discord ‚Üí Bot Node.js ‚Üí Seriale USB ‚Üí Arduino

    L'utente interagisce con l'interfaccia web che invia comandi tramite webhook al canale Discord. Il bot Node.js monitora il canale, interpreta i comandi e li invia ad Arduino tramite connessione seriale USB.

  2. Canale Joystick (Controllo Locale)

    Joystick ‚Üí Input Diretto Arduino

    Il joystick √® collegato direttamente ai pin analogici di Arduino e fornisce un controllo immediato senza passare attraverso il software intermedio.

Componenti Hardware e Collegamenti

Arduino Mega 2560

Microcontrollore con sufficienti pin I/O per gestire tutti i motori e sensori.

Motori Stepper (5x)

Per movimenti precisi di base, spalla e gomito. Controllati tramite driver dedicati.

Servomotori (2x)

Per il controllo del polso e della pinza con movimenti angolari precisi.

Joystick Analogico

Controllo manuale con due assi analogici e pulsante integrato.

Sensori Finecorsa (3x)

Per la calibrazione automatica all'avvio del sistema.

PC con Node.js

Esegue il bot Discord e gestisce la comunicazione seriale con Arduino.
