# Braccio Robotico Stampato in 3D e Controllato Multi-Piattaforma
Ho creato un braccio robotico partendo da 0.<br>
Un sistema che combina modellazione e stampa 3D, Arduino, Node.js e interfacce web e Discord per il controllo remoto di un braccio robotico completamente realizzato con la stampa 3D.<br>
